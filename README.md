# Automatic Landmark Prediction via Multi-Atlas SIFT-RANSAC

Ce projet implémente une méthode robuste de prédiction de landmarks anatomiques (AFIDs) basée sur le matching de features SIFT et une optimisation RANSAC multi-atlas.

## Prérequis

* Python 3.8+
* L'exécutable `featMatchMultiple` (non inclus dans ce repo)
